# DB 3D Product Model for WooCommerce

This is a WordPress plugin that adds 3D models to WooCommerce products. 

## Description

With the DB 3D Product Model for WooCommerce plugin, you can add 3D models to your WooCommerce products. The 3D models can be viewed directly on the product page, giving your customers a better understanding of your products.

## Installation

1. Download the plugin from the [Digital Buddha website](https://www.digitalbuddha.in).
2. Navigate to the WordPress admin dashboard, go to `Plugins > Add New > Upload Plugin`.
3. Upload the downloaded plugin file (should be a `.zip` file).
4. After the plugin has been uploaded, click `Activate` to start using the plugin.
5. Navigate to your product page, where you will find a new field under the general product data called '3D Model URL'.
6. Paste the URL of your 3D model file into this field and update the product.

## Usage

Once the plugin has been installed and activated:

1. Go to the edit screen for a WooCommerce product.
2. Under the `Product data` section, you will find a new field labeled `3D Model URL`.
3. Enter the URL of your 3D model in this field.
4. Update the product.

When you view the product, you should now see a 3D model displayed along with the product images.
